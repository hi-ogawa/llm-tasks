# Bass Tone Research

Exploring bass guitar tone shaping: clean amp sim, synth bass effects, and processing techniques.

## Gear

- **Bass**: Yamaha TRB1005J (5-string active, 3-band EQ)
- **Interface**: Focusrite Scarlett
- **DAW**: Ableton Live Lite (bundled with Scarlett)
- **Pedal**: Zoom B1 Four
- **Headphones**: AKG

## Files

- `notes/clean-tone.md` - Clean tone guide (amp sim + reverb)
- `notes/synth-bass.md` - Synth bass setup with octaver, filters, etc.
- `notes/ableton-setup.md` - Plugin management, latency, backing track blend
